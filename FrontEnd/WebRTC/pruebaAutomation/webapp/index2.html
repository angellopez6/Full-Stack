Bienvenido al cliente web de RDP

Con el cliente web de escritorio remoto ( RDP ) usted puede realizar llamadas con conexión punto a punto ( peer-to-peer ) y establecer una comunicación mediante un servidor TURN ( Traversal Using Relay NAT ) proveído gratuitamente por Google.

Ingrese el id del cliente

El id del cliente es indispensable para llamar al par remoto.
